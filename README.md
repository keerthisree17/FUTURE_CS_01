# Cyber Security Internship - Task 1

## Objective
Performed a passive vulnerability assessment on a public demo website using ethical and read-only testing techniques.

## Target Website
https://demo.owasp-juice.shop/

## Scope
- Passive scanning only
- No exploitation performed
- Read-only assessment

## Tools Used
- Nmap
- OWASP ZAP
- Browser DevTools
- Canva
- GitHub

## Findings
- Missing CSP Header
- Missing HSTS Header
- Cache-Control Issues
- Timestamp Disclosure
